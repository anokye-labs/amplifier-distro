2 Connect Your App

Enter the credentials from your app's Basic Information > App Credentials section, then install it to your workspace via OAuth.

3 App-Level Token

Generate an xapp- token to enable Socket Mode (no public URL needed). Follow the steps below in your Slack app settings.

  1. Go to Settings > Basic Information
  2. Scroll to App-Level Tokens
  3. Click Generate Token and Scopes
  4. Give it a name (e.g. socket-mode) and add scope connections:write
  5. Copy the token starting with xapp-
Open App Settings ↗
4 Choose Channel & Test

Pick a channel where the bot will listen for commands. We recommend creating #amplifier and running /invite @amplifier.
